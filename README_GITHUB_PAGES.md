# ResidenciAPP v35.11 · GitHub Pages

Versión lista para subir a GitHub Pages.

## Cómo subir

1. Descomprimir este ZIP.
2. Subir el contenido interno al repositorio público de publicación.
3. No subir el ZIP como archivo único.
4. En GitHub Pages usar: `Settings → Pages → Deploy from a branch → main → /root`.

Incluye `.nojekyll` para que GitHub Pages publique correctamente los archivos de `assets/`.
