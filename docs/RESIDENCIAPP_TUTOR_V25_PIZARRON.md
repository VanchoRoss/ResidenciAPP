# ResidenciAPP Tutor v2.5 · Pizarrón local mejorado

Cambios principales:

- La mini hoja personal pasa a funcionar como un único pizarrón local.
- Herramientas disponibles: birome, borrador por zonas, texto y mover cuadros.
- Paleta rápida de colores + selector de color libre.
- Grosor de trazo y tamaño de borrador configurables.
- Cuadros de texto insertables con click sobre el pizarrón.
- Los cuadros de texto se pueden mover, editar, resaltar, agrandar o eliminar.
- El pizarrón sigue siendo local: se guarda en localStorage de cada usuario.
- El pizarrón se revela recién después de contestar la pregunta.
