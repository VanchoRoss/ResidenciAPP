# ResidenciAPP v34.1 Premium Clean UX

Cambios aplicados sobre la v34 Banco 974:

- Inicio más limpio, con menos botones en el hero principal.
- Se conserva toda la funcionalidad desde menú lateral, accesos rápidos y modo pregunta.
- Se removió el módulo visible de Auditoría v34 del panel principal.
- Se removieron los botones de exportar base de datos JSON de la interfaz principal.
- El repaso espaciado ahora sale del bloque de vencidas al responder:
  - respuesta correcta: se reprograma automáticamente como Fácil (+4 días)
  - respuesta incorrecta: se reprograma automáticamente como Difícil (mañana)
  - si el usuario marca dificultad manualmente, esa elección sobrescribe la programación automática.
- En modo Revancha no se muestra el panel de rendimiento.
- En Retención avanzada se removió el botón redundante “¿Podés explicarlo?”.
