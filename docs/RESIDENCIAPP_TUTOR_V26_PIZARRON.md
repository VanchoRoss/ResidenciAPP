# ResidenciAPP Tutor v2.6 · Pizarrón local avanzado

Esta versión mejora la mini hoja personal de cada pregunta.

## Cambios principales

- Pizarrón en pantalla completa.
- Herramienta **Seleccionar** para mover cuadros de texto y trazos realizados con birome.
- Cuadros de texto más limpios: sin borde visible por defecto.
- Opción para activar borde de texto, elegir color y grosor.
- Color independiente para birome y texto.
- Resaltado de texto más preciso sobre letras seleccionadas.
- Borrador por zonas sobre trazos vectoriales nuevos.
- Guardado local por pregunta en el navegador del usuario.

## Nota técnica

Los trazos nuevos se guardan como objetos editables para poder moverlos. Los dibujos antiguos guardados como imagen de canvas pueden seguir viéndose, pero no siempre serán seleccionables como trazos individuales.
