# ResidenciAPP Tutor v2.8 · NeuroPREP funcional + Pizarrón consolidado

## Cambios principales

### Pizarrón local
- Botón de pantalla completa consolidado.
- Escape permite salir de pantalla completa.
- Cuadros de texto transparentes y sin borde por defecto.
- Modo seleccionar para mover texto y trazos de birome.
- Color propio para texto.
- Resaltado aplicado sobre letras seleccionadas.
- Borde opcional configurable en color y grosor.
- Borrador por zonas para trazos.

### NeuroPREP
- Diagnóstico rápido con hipótesis antes de opciones.
- Modo razonamiento guiado con confianza previa.
- Reflexión post-respuesta obligatoria/sugerida para analizar distractores.
- Registro local de sesiones NeuroPREP.
- Calibración entre confianza y acierto.
- Plan correctivo de 7 días según errores, repasos vencidos y temas débiles.
- Interleaving y examen predictivo integrados.

## Nota
El progreso, las hipótesis, reflexiones y pizarrón se guardan en el navegador de cada usuario. No se envían a Google Sheets.
