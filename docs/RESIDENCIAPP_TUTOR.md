# ResidenciAPP Tutor

La sección **Aprender desde cero** agrega una capa de enseñanza encima del banco de preguntas.

## Flujo recomendado

1. Elegir un nodo.
2. Leer la clase guiada.
3. Revisar algoritmo y trampas.
4. Marcar la lección como vista.
5. Iniciar preguntas relacionadas.
6. Guardar el nodo para repaso si quedó débil.

## Cómo agregar nuevos nodos

1. Crear un archivo HTML autónomo en `assets/lessons/`.
2. Agregar su metadata en `assets/data/lessons.js` y `assets/data/lessons.json`.
3. Definir `terms` para vincular preguntas del banco.

## Buen formato de nodo

- Concepto madre.
- Por qué importa en examen.
- Fisiopatología o lógica clínica.
- Diagnóstico.
- Conducta.
- Trampas frecuentes.
- Preguntas reales.
