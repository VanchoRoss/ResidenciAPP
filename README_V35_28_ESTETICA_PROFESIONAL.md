# ResidenciAPP v35.28 · Estética profesional

Versión enfocada exclusivamente en mejorar la experiencia visual completa de la app.

## Cambios principales

- Interfaz completa más sobria, seria y profesional.
- Paleta cálida y editorial inspirada en entornos de IA modernos: fondos marfil, superficies limpias, acento terracota y bajo ruido visual.
- Sidebar más ordenado y menos saturado.
- Header superior tipo workspace, con transparencia suave.
- Tarjetas, formularios, botones, paneles y modales unificados.
- Mejor contraste de lectura en preguntas, opciones y nodos.
- Aprender desde 0 integrado al nuevo sistema visual.
- Repaso, juegos, biblioteca, mapas y dashboard conservan sus funciones pero con estética más coherente.

## Seguridad de datos

Esta versión no modifica:

- preguntas;
- respuestas correctas;
- métricas;
- progreso;
- localStorage;
- reportes;
- endpoints de feedback;
- nodos de estudio.

Los cambios son principalmente CSS y una pequeña capa JS de arranque visual.
